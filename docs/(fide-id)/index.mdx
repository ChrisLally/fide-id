Fide ID defines the identifier method used to give entities stable, portable identifiers.

It is designed for identification, not control. A Fide ID identifies an entity from declared type information and a reference identifier, but it does not by itself prove authorship, authority, or ownership.

<Callout title="Specification">
  See [Specification](/fide-id/specification) for the identifier format, parts, and normative rules.
</Callout>

## Fide Ecosystem Alignment

- [Fide Entity Vocabulary](/vocabulary) defines the entity types and per-entity identifier constraints used by Fide ID
- [Fide Context Protocol](/fcp) uses Fide ID as its identifier method for statements and graph data
